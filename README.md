# homebrew-tap

Homebrew formulae by [@lyzlov-twelve](https://github.com/lyzlov-twelve).

## Usage

```bash
brew tap lyzlov-twelve/tap
brew install claude-sessions-tui
```

## Formulae

- **claude-sessions-tui** — Terminal UI for browsing Claude Code sessions from
  jsonl files. [Source](https://github.com/lyzlov-twelve/claude-sessions-tui).
